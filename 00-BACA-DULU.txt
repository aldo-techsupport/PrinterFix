 APA YANG BISA DIPERBAIKI OLEH TOOL INI?
═══════════════════════════════════════════════════════════════════════════

  ✓ Error 0x0000011b (RPC over SMB) - Paling umum!
  ✓ Error 0x000003e3 (Network Connection)
  ✓ Error 0x00000709 (Default Printer)
  ✓ Error 0x00000bcb (Driver Issues)
  ✓ Printer tidak bisa di-share
  ✓ Client tidak bisa connect ke printer
  ✓ Printer hilang setelah Windows Update
  ✓ Dan masalah printer sharing lainnya

═══════════════════════════════════════════════════════════════════════════
 CARA PENGGUNAAN SINGKAT
═══════════════════════════════════════════════════════════════════════════

 UNTUK SERVER (Komputer yang share printer):
 ───────────────────────────────────────────────────────────────────────────
   1. Double-click: PrinterSharingFix-AutoAdmin.bat
   2. Pilih: [1] SERVER
   3. Klik: "JALANKAN SEMUA PERBAIKAN"
   4. Backup: Yes (sangat direkomendasikan!)
   5. Restart komputer

 UNTUK CLIENT (Komputer yang akses printer):
 ───────────────────────────────────────────────────────────────────────────
   1. Double-click: PrinterSharingFix-AutoAdmin.bat
   2. Pilih: [2] CLIENT
   3. Klik: "JALANKAN SEMUA PERBAIKAN"
   4. Backup: Yes (sangat direkomendasikan!)
   5. Restart komputer
   6. Tambah printer: Fix 8 atau manual


═══════════════════════════════════════════════════════════════════════════
 PENTING! BACA INI
═══════════════════════════════════════════════════════════════════════════

  ⚠️  HARUS dijalankan sebagai Administrator
      (File AutoAdmin.bat akan otomatis meminta)

  ⚠️  SELALU backup sebelum perbaikan
      (Tool akan menawarkan backup otomatis)

  ⚠️  RESTART komputer setelah fix
      (Beberapa perubahan memerlukan restart)

  ⚠️  FIX SERVER dulu, baru CLIENT
      (Urutan penting untuk hasil optimal)

  ⚠️  SIMPAN backup di tempat aman
      (USB/Cloud untuk keamanan ekstra)


═══════════════════════════════════════════════════════════════════════════
 TROUBLESHOOTING CEPAT
═══════════════════════════════════════════════════════════════════════════

  ❓ Script tidak bisa jalan?
     → Klik kanan file .bat, pilih "Run as administrator"

  ❓ Masih error setelah fix?
     → Gunakan tombol "♻️ Restore dari Backup"

  ❓ Printer tidak muncul?
     → Gunakan "Fix 8: Tambah Printer Manual" di Client

  ❓ Tidak bisa connect?
     → Cek firewall, pastikan File and Printer Sharing enabled

  ❓ Error setelah Windows Update?
     → Jalankan fix lagi, biasanya langsung solved
