================================================================================
QUICK REFERENCE
================================================================================

UNTUK PEMULA:
1. Baca: MULAI-DISINI.txt
2. Jalankan: PrinterSharingFix-AutoAdmin.bat
3. Pilih: [1] Server atau [2] Client
4. Klik: "JALANKAN SEMUA PERBAIKAN"
5. Backup: Pilih "Yes" saat ditanya
6. Restart komputer


UNTUK TROUBLESHOOTING:
1. Jalankan: Test-Prerequisites.bat (cek sistem)
2. Baca: README.md (troubleshooting section)
3. Cek: Folder backup (Documents\Backup Printer)
4. Restore: Jika perlu, gunakan tombol Restore di tool


UNTUK BACKUP/RESTORE:
1. Baca: BACKUP-INFO.txt
2. Backup: Tombol "💾 Backup Registry Sekarang"
3. Restore: Tombol "♻️ Restore dari Backup"
4. Lokasi: Documents\Backup Printer

ERROR CODES YANG DITANGANI
================================================================================

✓ 0x0000011b  - RPC over SMB (paling umum setelah Windows Update)
✓ 0x000003e3  - Network Connection Error
✓ 0x00000709  - Default Printer Error
✓ 0x00000bcb  - Driver Issues
✓ 0x00004005  - Access Denied
✓ 0x00000002  - Printer Not Found
✓ Dan error printer sharing lainnya


================================================================================
SUPPORT & HELP
================================================================================

Jika mengalami masalah:

1. Cek Prerequisites
   Jalankan: Test-Prerequisites.bat

2. Baca Troubleshooting
   File: README.md (bagian Troubleshooting)

3. Cek Log
   Log ada di GUI tool saat menjalankan fix

4. Restore Backup
   Jika fix menyebabkan masalah, restore dari backup

5. Manual Fix
   Baca: BACKUP-INFO.txt untuk cara restore manual
