LOKASI BACKUP
-------------
Semua backup disimpan di:
C:\Users\[NamaUser]\Documents\Backup Printer

Contoh:
C:\Users\John\Documents\Backup Printer


FORMAT FILE BACKUP
------------------
Server: PrinterRegistry_Server_YYYYMMDD_HHMMSS.reg
Client: PrinterRegistry_Client_YYYYMMDD_HHMMSS.reg

Contoh:
PrinterRegistry_Server_20241022_143025.reg
PrinterRegistry_Client_20241022_143530.reg


APA YANG DI-BACKUP?
-------------------
Tool ini mem-backup registry keys berikut:

1. HKLM\System\CurrentControlSet\Control\Print
   - Berisi konfigurasi Print Spooler
   - Setting RPC over SMB
   - Konfigurasi printer driver

2. HKLM\SOFTWARE\Policies\Microsoft\Windows NT\Printers
   - Point and Print policies
   - Printer sharing permissions
   - Security settings


KAPAN HARUS BACKUP?
-------------------
✓ SEBELUM menjalankan perbaikan pertama kali
✓ SEBELUM melakukan perubahan besar pada sistem
✓ Setelah konfigurasi printer bekerja dengan baik (sebagai backup "golden state")
✓ Sebelum Windows Update yang besar


KAPAN HARUS RESTORE?
--------------------
✗ Setelah perbaikan, printer malah tidak bisa digunakan
✗ Muncul error baru yang sebelumnya tidak ada
✗ Sistem menjadi tidak stabil
✗ Ingin kembali ke konfigurasi sebelumnya


CARA BACKUP MANUAL
------------------
1. Jalankan PrinterSharingFix-AutoAdmin.bat
2. Pilih [1] untuk Server atau [2] untuk Client
3. Klik tombol "💾 Backup Registry Sekarang"
4. File backup akan tersimpan otomatis


CARA RESTORE
------------
1. Jalankan PrinterSharingFix-AutoAdmin.bat
2. Pilih [1] untuk Server atau [2] untuk Client (sesuai backup yang ingin di-restore)
3. Klik tombol "♻️ Restore dari Backup"
4. Pilih file backup dari daftar (biasanya yang paling baru)
5. Klik "Restore" dan konfirmasi
6. Restart komputer


TIPS PENTING
------------
• Simpan backup di tempat lain (USB/Cloud) untuk keamanan ekstra
• Jangan hapus backup lama, disk space yang digunakan sangat kecil
• Catat tanggal backup yang "working" untuk referensi
• Backup terpisah untuk Server dan Client, jangan tertukar!


TROUBLESHOOTING BACKUP
-----------------------
Q: Folder backup tidak ada?
A: Tool akan membuat folder otomatis saat backup pertama kali

Q: Tidak bisa restore?
A: Pastikan menjalankan tool sebagai Administrator

Q: File backup corrupt?
A: Gunakan backup yang lebih lama dari daftar

Q: Backup tidak muncul di daftar restore?
A: Pastikan file ada di folder: Documents\Backup Printer
   Dan nama file sesuai format: PrinterRegistry_[Server/Client]_*.reg


RESTORE MANUAL (Jika Tool Tidak Bisa Digunakan)
------------------------------------------------
1. Buka File Explorer
2. Pergi ke: C:\Users\[NamaUser]\Documents\Backup Printer
3. Klik kanan pada file .reg yang ingin di-restore
4. Pilih "Merge" atau "Import"
5. Klik "Yes" untuk konfirmasi
6. Restart komputer


KONTAK & SUPPORT
-----------------
Jika mengalami masalah dengan backup/restore:
1. Pastikan menjalankan sebagai Administrator
2. Cek log di tool untuk pesan error
3. Coba restore manual dengan cara di atas
4. Pastikan file backup tidak corrupt (cek ukuran file > 0 KB)
