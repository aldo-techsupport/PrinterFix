┌────────────────────────────────────────────────────────────────────────────┐
│ CARA TERCEPAT                                                              │
└────────────────────────────────────────────────────────────────────────────┘

  1. Double-click: PrinterSharingFix-AutoAdmin.bat
  2. Pilih: [1] Server atau [2] Client
  3. Klik: "JALANKAN SEMUA PERBAIKAN"
  4. Backup: Yes
  5. Restart


┌────────────────────────────────────────────────────────────────────────────┐
│ ERROR CODES                                                                │
└────────────────────────────────────────────────────────────────────────────┘

  0x0000011b  →  RPC over SMB (paling umum)
  0x000003e3  →  Network Connection
  0x00000709  →  Default Printer
  0x00000bcb  →  Driver Issues

┌────────────────────────────────────────────────────────────────────────────┐
│ BACKUP & RESTORE                                                           │
└────────────────────────────────────────────────────────────────────────────┘

  Lokasi Backup:
    C:\Users\[Username]\Documents\Backup Printer

  Backup:
    • Otomatis ditawarkan sebelum fix
    • Manual: Tombol "💾 Backup Registry Sekarang"

  Restore:
    • Tombol "♻️ Restore dari Backup"
    • Pilih backup terakhir
    • Restart komputer

┌────────────────────────────────────────────────────────────────────────────┐
│ TROUBLESHOOTING                                                            │
└────────────────────────────────────────────────────────────────────────────┘

  Masalah                          Solusi
  ─────────────────────────────────────────────────────────────────────────
  Script tidak jalan               → Klik kanan "Run as administrator"
  Masih error setelah fix          → Restore dari backup
  Printer tidak muncul             → Fix 8: Tambah Printer Manual
  Tidak bisa connect               → Cek firewall & network
  Error setelah Windows Update     → Jalankan fix lagi


┌────────────────────────────────────────────────────────────────────────────┐
│ COMMAND REFERENCE                                                          │
└────────────────────────────────────────────────────────────────────────────┘

  Tambah Printer Manual:
    \\SERVER-NAME\PRINTER-NAME

  Cek Print Spooler:
    services.msc → Print Spooler

  Cek Firewall:
    Control Panel → Windows Defender Firewall
    → Allow an app → File and Printer Sharing


┌────────────────────────────────────────────────────────────────────────────┐
│ TIPS                                                                       │
└────────────────────────────────────────────────────────────────────────────┘

  ✓ Selalu backup sebelum fix
  ✓ Fix SERVER dulu, baru CLIENT
  ✓ Restart setelah setiap fix
  ✓ Catat nama printer yang working
  ✓ Simpan backup di USB/Cloud juga
  ✓ Test print setelah fix


┌────────────────────────────────────────────────────────────────────────────┐
│ KEYBOARD SHORTCUTS (Di GUI)                                               │
└────────────────────────────────────────────────────────────────────────────┘

  Alt+F4          →  Tutup aplikasi
  Ctrl+C          →  Copy dari log
  Ctrl+A          →  Select all log


┌────────────────────────────────────────────────────────────────────────────┐
│ CHECKLIST SEBELUM FIX                                                      │
└────────────────────────────────────────────────────────────────────────────┘

  □ Jalankan sebagai Administrator
  □ Print Spooler service berjalan
  □ Network terhubung
  □ Tahu nama server dan printer
  □ Siap restart komputer
  □ Backup penting sudah disimpan


┌────────────────────────────────────────────────────────────────────────────┐
│ CHECKLIST SETELAH FIX                                                      │
└────────────────────────────────────────────────────────────────────────────┘

  □ Restart komputer
  □ Cek Print Spooler berjalan
  □ Test print
  □ Simpan backup di tempat aman
  □ Catat konfigurasi yang working


┌────────────────────────────────────────────────────────────────────────────┐
│ EMERGENCY RESTORE                                                          │
└────────────────────────────────────────────────────────────────────────────┘

  Jika tool tidak bisa dibuka:

  1. Buka: C:\Users\[Username]\Documents\Backup Printer
  2. Klik kanan file .reg terakhir
  3. Pilih: "Merge"
  4. Klik: "Yes"
  5. Restart komputer
